O objetivo do algorítmo é exibir ao utilizador os 5 maiores investidores por distrito (Lisboa, Porto, Faro, Braga e Aveiro),
seguido de outras exibições dos 3 maiores investimentos por cada um desses investidores na região pedida.
Nota: 

Requisitos:

Será necessário existir um servidor MySQL com o dataset importado, para fazer a conexão com o código python.
Antes de correr, seja via IDE, Notebook ou linha de comandos é necessário mudar as configurações da conexão à base de dados, alterando o host, username, password e base de dados


Resumo do código:

O código dispõe de 2 classes, uma que faz ligação com a base de dados e executa queries pedidas à mesma base de dados; a outra classe trata dos dados pedidos à base de dados e
trata de os dispôr nas lista pretendidas

EN

Python algorithm that displays a graphical interface, with backend connection to the database.
The goal of the algorithm is to display to the user the 5 largest investors by district (Lisbon, Porto, Faro, Braga and Aveiro),
followed by other displays of the 3 largest investments by each of these investors in the requested region.
Note: 

Requirements:

There will need to be a MySQL server with the dataset imported, to make the connection to the python code.
Before running, either via IDE, Notebook or command line it is necessary to change the database connection settings by changing the host, username, password and database


Summary of the code:

The code has 2 classes, one that connects to the database and executes queries requested to the same database; the other class handles the data requested to the database and
arranges them in the desired lists.
